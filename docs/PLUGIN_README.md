# CCPM Plugin Quick Start

Ship faster with spec-driven development, GitHub Issues, Git worktrees, and parallel AI agents.

## Install

```bash
/plugin install /path/to/ccpm
```

## Quick Commands

```bash
/ccpm:status          # Project dashboard
/ccpm:prd-new         # Create PRD
/ccpm:epic-start      # Start epic worktree
/ccpm:issue-start     # Work on issue
/ccpm:help            # All commands
```

## Workflow

```mermaid
graph LR
    A[PRD] --> B[Epic]
    B --> C[Tasks]
    C --> D[GitHub]
    D --> E[Parallel Work]
```

1. **Brainstorm** → `/ccpm:prd-new`
2. **Plan** → `/ccpm:prd-parse`
3. **Decompose** → `/ccpm:epic-decompose`
4. **Sync** → `/ccpm:github-sync`
5. **Execute** → `/ccpm:epic-parallel`
